# Android Auto Clicker (Self-learning Project)

## Project Description
This is a self-learning Android project aiming to implement a basic auto clicker system.

## Current Features
- Basic UI with button interaction
- ClickPoint data structure
- List-based click point management

## Development Progress
- v0.1: Initial project setup
- v0.2: UI interaction implemented
- v0.3: ClickPoint data structure added

## Future Plans
- Add / remove click points
- Implement accessibility service
- Execute automated taps
